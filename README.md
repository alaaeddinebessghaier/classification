# Classification Image Project

## Installation

1. Clone the repo:
   git clone https://github.com/<username>/classification.git
2. Create a virtual environment:
   python -m venv venv
3. Activate it:
   - Windows: venv\Scripts\activate
   - Mac/Linux: source venv/bin/activate
4. Install dependencies:
   pip install -r requirements.txt

## Running the App

1. From the project root:
   python -m app.app
2. Open a browser at:
   http://127.0.0.1:5000
3. Upload an image, select a model (KNN, DecisionTree, NaiveBayes), and view predictions.
